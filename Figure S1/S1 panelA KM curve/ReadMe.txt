This code will generate the figure seen in figure S1 panel A. The figure is generated by creating a Kaplan Meier surival plot . The input file for this analysis is provided in this directory. The input files were generated from publicly available data available at TCGA (reference to location) and METABRIC (reference to location). If you do choose to run the R program in this directory, please change the working directory in the code to the location of the input files.

The output for this script is the generation of a Kaplan Meier plot.
